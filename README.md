# 🌙 Dreams

🚀 Repositório: https://github.com/OnisaikoDev/Dreams  

---

## 💻 Sobre o Projeto

O **Dreams** é um projeto desenvolvido com foco na criação de uma interface visual com temática relacionada a sonhos, explorando design, cores e organização de elementos na tela.

A proposta do projeto é trabalhar a construção de layouts modernos e expressivos, utilizando conceitos de UI para transmitir uma experiência mais imersiva.

---

## 🛠️ Tecnologias Utilizadas

- HTML5  
- CSS3  

---

## ✨ Características

- 🎨 Interface com identidade visual temática  
- 📐 Organização de layout e elementos  
- 📱 Estrutura adaptável para diferentes telas  
- ⚡ Foco em design e experiência visual  

---

## 📈 Objetivo

Este projeto foi desenvolvido com o objetivo de:

- Praticar criação de interfaces visuais  
- Explorar criatividade no desenvolvimento front-end  
- Melhorar habilidades em HTML e CSS  
- Construir projetos com identidade própria
